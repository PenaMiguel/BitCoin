# BitCoin
BitCoin is an iOS app that display and monitor the current Bitcoint (BTC) rate.


# Requirements

- Xcode 9 or later

### Description
BitCoin is an iOS app that display and monitor the current Bitcoint (BTC) rate.

Track real-time rate from Bitcoin (BTC).

Moreover, you can analysis price history by week, month, year, and from the beginning.
